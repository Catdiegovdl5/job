## 🎨 Estética e Design (Hacker Growth)

Em vez de usar templates genéricos, desenvolvi uma **Identidade Visual Absoluta**:
- UI inspirada em terminais de comando (Terminal/Cyberpunk).
- Animações microscópicas (`hover states`, partículas de terminal, glitches controlados) para **retenção de atenção**.
- Foco extremo na legibilidade (Fontes como *JetBrains Mono*, *Bebas Neue* e fundos escuros absolutos).
- O design atua como um filtro: atrai empresas inovadoras que valorizam profissionais fora da caixa.
